# Text Extraction
It is a wrapper and combiner of [Stanford CoreNLP](https://stanfordnlp.github.io/CoreNLP/), [OpenNLP](https://opennlp.apache.org/), [OpenIE](https://github.com/knowitall/openie), [ClausIE](https://www.mpi-inf.mpg.de/departments/databases-and-information-systems/software/clausie/), [WordNet](https://wordnet.princeton.edu/) and [ConceptNet API](https://github.com/commonsense/conceptnet5/wiki/API) to make those tools easily to use for tackling some NLP tasks, like **Named Entity Recognition**, **POS Tagging**, **Chunking**, **Information Extraction**, **Dependency Parising**, **Concept Extraction** and etc.

_(Currently, It's a very beginning version, a lot of function is preparing......)_
